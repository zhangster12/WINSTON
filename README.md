# WINSTON

## Installation


```
pip install speechRecognition
pip install pyttsx3
pip install pyaudio
```

If `pip install pyaudio` doesn't work:
```
pip install pipwin
pipwin install pyaudio
```

### Run script:
```
py winston.py
```


### Activate virtual environment:
```
vir_env/Scripts/activate
```

### Leave virtual environment:
```
deactivate
```
